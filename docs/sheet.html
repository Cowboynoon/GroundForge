GroundForge

Pricking prototypes for bobbin lace grounds.

When arriving on this page from the TesseLace index, there may be multiple patterns which can all be morphed to match the same pair diagram as shown and explained in the user guide.

Use the variant(s) you happen to like, or download them and use a third party editor to create your own variants. Follow the link above no matter which of the patterns to choose stitches for working diagrams.
