GroundForge - Diagrams of bobbin lace patterns

Make variations on predefined patterns

  • Select a pattern from a catalogue page: the diagram links in the image captions of Tesselace index and flanders return to this page.
  • Make variations on the selected pattern by choosing stitches in the prototype diagram (clear the stitches)
  • Choose contrasting threads in the thread diagram.
  • Start the animation of the pair diagram to make the holes as round as in the thread diagram.
