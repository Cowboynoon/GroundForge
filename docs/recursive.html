GroundForge

Uses a thread diagram as pair diagram to create another thread diagram.

Create Binche, Flanders or paris from a plain Torchon (respectively ctctc, ctct or ct with the first thumbnail), and many more.

Numbers

When limiting the choices to the three most common stitches we get 3+9+27 thread diagrams for one start pattern. Extending the choice of stitches with ctctc with or without an additional twist (frequently used in traditional Alpine free-hand lace where normal Torchon uses pins), the numbers become 5+25+125. No repeated twist are possible, so further variations can only drop a left or right twist or make the Alpine mini plait longer.

Usage

The main page explains the paramters on the matrix tab, patch size tab and footside tab.

  • Select a thumbnail to set the parameters or copy values from the main page or thumbnail page.
  • Show the diagrams top to bottom with the desired stitches.
