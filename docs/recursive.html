GroundForge

Uses a thread diagram as pair diagram to create another thread diagram.

When limiting the choices to the three most common stitches we get 3+9+27 thread diagrams for one start pattern, add the ctctc frequently used in free-hand lace, the numbers become 4+16+64.

First select the start pattern, then show the diagrams top to bottom with the desired stitches. The main page explains the options in more details.
