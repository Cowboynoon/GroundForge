GroundForge

A toolbox to design bobbin lace grounds with matching diagrams.

The diagrams and patches on the pattern sheet match up as a tool set to design a bobbin lace ground. You can create variations with the following steps, the user guide provides more tips and tricks.

  • Select a thumbnail above, or use the advanced options to configure even more patterns.
  • Stitches with an odd number of crossings cause threads to follow other paths than pairs, rendering different results. Use the form to define where to apply which stitch. Hover over a stitch in the color coded pair diagram to discover its id. You can choose a bigger patch if your system is fast enough, or zoom and pan a diagram.
  • Decide where to apply a contrasting color or thread size: select the square start of a thread to paint it and see its path.
  • Nudging the pins to slightly different positions can have a significant effect. Download the pattern sheet, it contains reshaped variants of the pair diagram. Customize a patch to your own taste and needs with an SVG editor.
